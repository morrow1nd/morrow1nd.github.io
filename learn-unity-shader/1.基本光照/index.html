1.基本光照

Phone光照模型

cambient = gambient

cemissive = memissive

cdiffuse = (clight * mdiffuse)max(0, n * l)

cspecular = (clight * mspecular)max(0, v * r)mgloss

r = 2(n * l)n - l

其中r是反射方向。 在Unity中可以使用内置函数reflect计算。

半兰伯特(Half Lambert)模型

cdiffuse = (clight * mdiffuse)(a(n * l) + b)

Blinn-Phong光照模型

h = (v + l) / | v + l | = normalize(v + l)
